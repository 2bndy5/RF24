examples_linux/streaming_data.py

This is a simple example of using the RF24 class on a Raspberry Pi for streaming multiple payloads.

Remember to install the Python wrapper, then navigate to the "RF24/examples_linux" folder and open a python3 REPL.
To run this example, enter

1 from streaming_data import *

and follow the prompts.

Note
this example requires python v3.7 or newer because it measures transmission time with time.monotonic_ns().

56 def make_buffer(buf_iter, size=32):
57  """return a list of payloads"""
58  # we'll use `size` for the number of payloads in the list and the
59  # payloads' length
60  # prefix payload with a sequential letter to indicate which
61  # payloads were lost (if any)
62  buff = bytes([buf_iter + (65 if 0 <= buf_iter < 26 else 71)])
63  for j in range(size - 1):
64  char = bool(j >= (size - 1) / 2 + abs((size - 1) / 2 - buf_iter))
65  char |= bool(j < (size - 1) / 2 - abs((size - 1) / 2 - buf_iter))
66  buff += bytes([char + 48])
