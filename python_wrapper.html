Python Wrapper Installation¶

wrapper was initially written by mz-fuzzy

Warning

Python2 is no longer supported

Prerequisites Prerequisites¶

Python3¶

The python examples need Python v3.7 because of the time.monotonic_ns() call for measuring microseconds.

sudo apt-get install python3-dev libboost-python-dev python3-setuptools python3-rpi.gpio

RF24 (C++ library)¶

The RF24 lib needs to be built in C++ & installed for the python wrapper to wrap it

See Linux Installation and Linux/RPi General

Install Installation:¶

  1. For python3 in Raspberry Pi OS (formerly known as Raspbian), it’s needed to manually link python boost library, like this:

    sudo ln -s $(ls /usr/lib/arm-linux-gnueabihf/libboost_python3-py3*.so | tail -1) /usr/lib/arm-linux-gnueabihf/libboost_python3.so

  2. Build the library. From the rf24libs/RF24/pyRF24 directory:

    python3 setup.py build

    Note

    Build takes several minutes on arm-based machines. Machines with less than 1 GB of RAM may need to increase amount of swap for build.

  3. Install the library

    sudo python3 setup.py install

    See the included *.py files in the “examples_linux” folder and the RF24 documentation for usage information.

  4. Running the Example:

    See the additional Linux General page for information on connecting your hardware.

    Edit the getting_started.py example to configure the appropriate pins per the link:

    nano getting_started.py

    Configure another device with one of:

    • getting_started.py Python example

    • GettingStarted.ino Arduino example

    • gettingStarted.cpp Linux example

    Run the example

    sudo python3 getting_started.py
